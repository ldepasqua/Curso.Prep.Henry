Variables: son lugares donde yo puedo guardar algo que posteriormente puedo visualizar. Puedo crear cuantas variables quiera y a cada una de ellas les tengo que poner un nombre único que las identifique. La información que pueden guardar son de tipo string/texto, números o booleanos (verdadero o falso), también pueden no estar definidas o ser nullas (vacías).

Strings: Una serie de caracteres que pueden ser números, letras, simbolos, etc. Se encuentran encerrados entre comillas simples. 

Funciones: Es una cajita que agrupa código que la guardo en un lugar y la puedo usar cuando quiera. Me ahorra tener que escribir el mismo código muchas veces. Estas funciones pueden contar con argumentos que funcionan como variables para la funcion. También existen las funciones return, las que "no retornan nada" por defecto retornan undefined y las que retornan si devuelven algo distinto a undefined. Las funciones return solo retornan una única vez, todo código que esté por debajo del return no va a ser ejecutado después de retornar.

Declaraciones: Es la creación o utilización de variables dentro de una funcion. En el caso del if ya es parte de una lógica, establezco una condicion que si se cumple (es verdadera) entonces se va a ejecutar el codigo encerrado dentro del if, y si la condición no se cumple no se ejecuta el código o se pasa a un else donde se ejecuta otro código. Esto se puede repetir la cantidad de veces que quiera con un elseif siempre y cuando cuente con condiciones para seguir evaluando.

Valores booleanos: Los valores booleanos pueden ser asignados a una variable, pueden ser el resultado de la evaluación de una condicion o pueden ser el resultado de una función a traves de un return.
